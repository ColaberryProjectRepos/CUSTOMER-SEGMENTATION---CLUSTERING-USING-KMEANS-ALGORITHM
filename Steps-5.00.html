STEP 5: Model Selection

·     ·          Algorithm: KMEANS

·     ·           KMEANS Algorithm hyper parameters (input taken) are:

·     ·          n_clusters (Total no of clusters to form as well as the number of centroids to generate).

·     ·          init (Method for initialization, k-means++ is used to select the initial centroids to converge in a faster way).

·     ·          Features to fit the algorithm (Annual income and spending score are passed).

·     ·         Clustering is an unsupervised learning technique used to cluster unlabelled data.

·     ·         K-means is a type of unsupervised learning and one of the popular methods of clustering unlabelled data into k clusters.

·    ·         No of clusters can be given manually if there is an idea on exactly on how many exact number of clusters required.

      ·         If the number of clusters required is unknown, ELBOW Method is used to return the optimal number of clusters.

       

         

      ·        To determine the optimal number of clusters, we have to select the value of k at the “elbow” i.e. the point after which the inertia start decreasing in a linear fashion. 

·    ·         Here in this case optimal number of clusters are 5.
